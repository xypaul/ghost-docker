# Ghost Docker

Configuration to run Ghost and its services with Docker Compose.

This is a fork of [TryGhost/ghost-docker](https://github.com/TryGhost/ghost-docker)
adapted for deployment on [Coolify](https://coolify.io/) (Traefik proxy)
rather than the upstream's bundled Caddy. See [`COOLIFY.md`](./COOLIFY.md) for
the Coolify-specific deployment guide, required env vars, and gotchas
encountered in production.
